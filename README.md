# m-date-picker

html5 m-date-picker

## 演示地址

https://pmg1989.github.io/m-date-picker

## 快速开始

克隆项目文件:

```
git clone git@github.com:pmg1989/m-date-picker.git
```

进入目录安装依赖:

```
npm install 或者 yarn 或者 yarn install
```

开发：

```bash
run npm start or yarn start or gulp connect

打开 http://localhost:8888
```
